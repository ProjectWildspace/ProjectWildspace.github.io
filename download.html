How to Download

Step 1: Download the Installer

Click the button below to download the installer for Project Wildspace. It will redirect you to an OrangeDox link, which we use to keep a count of the number of downloads to measure scale.

Download

Step 2: Run the Installer

Follow the instructions of the installer to install Project Wildspace. It will install the program in the default location, and create a shortcut on your desktop. Be sure to pay attention to the filepath, as you will need to navigate there later.

Step 3: Setup OpenAI API Key

Please navigate to the link below to create an OpenAI account and generate an API key. Be sure to configure your billing options to ensure the software works. Do not reveal this API key. (Note: In terms of pricing, we have tested roughly 100 cases and are only down 12 cents)

OpenAI

Step 4: Configure your api.txt file

Find the program directory from the filepath specified in the installer. There, you must make a text (.txt) file named "api" (case sensitive). In this file, you must paste your OpenAI API key.

Step 5: Run the Program!

To experience full functionality of Project Wildspace, it is recommended to create world files first, although it is not required. You may generate a plethora of content, contextualized from these world files. Also, you may save any generation you are particularly fond of, and they will be stored in the saves folder of the program directory.
